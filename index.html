    The Game Exchange

    The Game Exchange is a platform that allows video game players in the community to meet up and exchange games with each other, no cash involved. Built with Python, Flask, Postgres, Javascript, jQuery, HTML, CSS.

    URL: https://the-game-exchange.herokuapp.com

    Code: https://github.com/kmalfatti/game-exchange

    winio

    I collaborated with two other Web Developers to built a wine pairing app in four days, which allows the user to select from a list of food items, and displays the wine type that pairs they best with that food. My role were research, UX design, and algorithm design. Built with Javascript, jQuery, CSS, Jade, Node.js, Express, Knex, PostreSQL, and Passport.

    URL: https://winio.herokuapp.com

    CODE: https://github.com/kmalfatti/q2_wine_app

    MGS3 Memory Game

    MGS3 Memory Game is a simple card flipping game, which displays 16 cards, and the user must find all eight unique pairs. Its theme is based on one of my favorite video games, Metal Gear Solid 3. Built only with HTML, CSS, Javascript, and Lodash.

    URL: http://mgs3memorygame.bitballoon.com

    Code: https://github.com/kmalfatti/memory-game

    Guess That Meme

    Guess That Meme is a guessing game that utilizes an API to display a popular meme without any text. The user must enter the name or phrase associated with that meme. Built with HTML, CSS, Bootstrap, and Javascript.

    URL: http://guessthatmeme.bitballoon.com

    Code: https://github.com/kmalfatti/guess-that-meme
